### What functionality does your code sample abstract away?

This feature abstracts away an event carousel component which is used on the landing page to display all events (in the future, featured events) and on a user's home page, all the events that they have signaled interest in, and allows different views of these events.

### How does it mechanically achieve abstraction?

The `ScheduleView` componenet abstracts away the various layouts of an event into the `EventCard` component. Then based on the view that the implementation requires (i.e. "simple" or "detailed" or "calendar view"), displays the `EventCard` component with the required alterations.

### What purpose does this abstraction serve in your larger program?

The purpose was to be able to display a list of upcoming events for different contexts, without having to duplicate code that is shared across those contexts like the standard layout we have for eac event.
